Convert from one ID to another

URL/rest/convert/{from}/{to}/{query}
MethodGET
URL Params

from — The source ID type (see source identifier types)

to — The target ID type (see target identifier types)

query — The identifier to convert

Sample Call 
curl https://cts.fiehnlab.ucdavis.edu/rest/convert/InChIKey/Chemical%20Name/QNAYBMKLOCPYGJ-REOHCLBHSA-N
Sample Response 
[
  {
    "fromIdentifier": "InChIKey",
    "searchTerm": "QNAYBMKLOCPYGJ-REOHCLBHSA-N",
    "toIdentifier": "Chemical Name",
    "result": ["L-Alanine"]
  }
]

Get a list of scored InChIKeys

URL/rest/score/{from}/{query}/{algorithm}
MethodGET
URL Params

from — The source ID type

query — The identifier to convert

algorithm — Scoring algorithm: biological or popularity

Sample Call 
curl https://cts.fiehnlab.ucdavis.edu/rest/score/Chemical%20Name/threonine/biological
Sample Response 
{
  "searchTerm": "threonine",
  "from": "chemical name",
  "result": [
    { "InChIKey": "AYFVYJQAPQTCCC-GBXIJSLDSA-N", "score": 1 },
    { "InChIKey": "AYFVYJQAPQTCCC-STHAYSLISA-N", "score": 0.86 },
    { "InChIKey": "AYFVYJQAPQTCCC-UHFFFAOYSA-N", "score": 0 }
  ]
}

Expand a chemical formula

URL/rest/expandformula/{formula}
MethodGET
URL Params

formula — The chemical formula to expand

Sample Call 
curl https://cts.fiehnlab.ucdavis.edu/rest/expandformula/H20
Sample Response 
{
  "formula": "H2O",
  "result": "HHO"
}

Convert an InChIKey to MOL (MDL/SDF)

URL/rest/inchikeytomol/{inchikey}
MethodGET
URL Params

inchikey — The InChIKey to convert

Sample Call 
curl https://cts.fiehnlab.ucdavis.edu/rest/inchikeytomol/QNAYBMKLOCPYGJ-REOHCLBHSA-N
Sample Response 
{
  "molecule": "\\n  CDK  ...\\nM  END\\n",
  "message": null
}

Count biological IDs

URL/rest/countBiological/{inchikey}
MethodGET
URL Params

inchikey — The InChIKey to count

Sample Call 
curl https://cts.fiehnlab.ucdavis.edu/rest/countBiological/QNAYBMKLOCPYGJ-REOHCLBHSA-N
Sample Response 
{
  "KEGG": 2,
  "BioCyc": 1,
  "Human Metabolome Database": 1,
  "total": 4
}
