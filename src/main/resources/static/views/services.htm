Services

Here you can find an overview of additional services the CTS offers.

Shortcuts

  • Convert from one ID to another
  • Get a list of scored InChIKeys
  • Get a list of valid source IDs
  • Get a list of valid target IDs
  • Expand a chemical formula
  • Convert an InChIKey to a molecule definition (MDL/SDF)
  • Count biological IDs

Convert from one ID to another

Title Convert
URL /rest/convert/{from}/{to}/{query} or /service/convert/{from}/{to}/{query}
Method GET
URL Params

from=[string]

The source ID type (for valid IDs, see /rest/fromValues)

to=[string]

The target ID type (for valid IDs, see /rest/toValues)

query=[string]

The id to convert
Sample Call 
curl http://cts.fiehnlab.ucdavis.edu/rest/convert/InChIKey/Chemical%20Name/QNAYBMKLOCPYGJ-REOHCLBHSA-N
Sample Response 
[
    {
        "fromIdentifier": "InChIKey",
        "searchTerm": "QNAYBMKLOCPYGJ-REOHCLBHSA-N",
        "toIdentifier": "Chemical Name",
        "result":
        [
            "L-Alanine"
        ]
    }
]

Get a list of scored InChIKeys

Title Score
URL /rest/score/{from}/{query}/{algorithm} or /service/score/{from}/{query}/{algorithm}
Method GET
URL Params

from=[string]

The source ID type

query=[string]

The ID to convert

algorithm=[string]

The scoring algorithm to use, either 'biological' or 'popularity'
Sample Call 
curl http://cts.fiehnlab.ucdavis.edu/rest/score/Chemical%20Name/threonine/biological
Sample Response 
{
    "searchTerm": "threonine",
    "from": "chemical name",
    "result":
    [
        {
            "InChIKey": "AYFVYJQAPQTCCC-GBXIJSLDSA-N",
            "score": 1
        },
        {
            "InChIKey": "AYFVYJQAPQTCCC-STHAYSLISA-N",
            "score": 0.86
        },
        {
            "InChIKey": "AYFVYJQAPQTCCC-UHFFFAOYSA-N",
            "score": 0
        }
    ]
}

Count biological IDs

Title Count Biological
URL /rest/countBiological/{inchikey} or /service/countBiological/{inchikey}
Method GET
URL Params

inchikey=[string]

The InChIKey to count
Sample Call 
curl http://cts.fiehnlab.ucdavis.edu/rest/countBiological/QNAYBMKLOCPYGJ-REOHCLBHSA-N
Sample Response 
{
    "KEGG": 2,
    "BioCyc": 1,
    "Human Metabolome Database": 1,
    "total": 4
}
