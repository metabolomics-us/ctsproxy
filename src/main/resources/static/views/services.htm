Services

Advanced users can access the CTS directly via REST endpoints. These include:

Convert

Convert an input string ("string") from one identifier ("from") to another ("to").

REST endpoint:

/rest/convert/[from]/[to]/[string]

Sample call:

curl http://cts.fiehnlab.ucdavis.edu/rest/convert/InChIKey/Chemical%20Name/QNAYBMKLOCPYGJ-REOHCLBHSA-N

Expected result:

[
    {
        "fromIdentifier": "InChIKey",
        "searchTerm": "QNAYBMKLOCPYGJ-REOHCLBHSA-N",
        "toIdentifier": "Chemical Name",
        "result":
        [
            "L-Alanine"
        ]
    }
]

Score

Get scored InChIKeys for a string ("string") of a given identifier type ("from") using your choice of algorithm ("algorithm", either biological or popularity).

REST endpoint:

/rest/score/[from]/[string]/[algorithm]

Sample call:

curl http://cts.fiehnlab.ucdavis.edu/rest/score/Chemical%20Name/threonine/biological

Expected result:

{
    "searchTerm": "threonine",
    "from": "chemical name",
    "result":
    [
        {
            "InChIKey": "AYFVYJQAPQTCCC-GBXIJSLDSA-N",
            "score": 1
        },
        {
            "InChIKey": "AYFVYJQAPQTCCC-STHAYSLISA-N",
            "score": 0.86
        },
        {
            "InChIKey": "AYFVYJQAPQTCCC-UHFFFAOYSA-N",
            "score": 0
        }
    ]
}

Values

Fetch a list of valid IDs for conversion.

REST endpoints:

/rest/fromValues
/rest/toValues

Sample calls:

curl http://cts.fiehnlab.ucdavis.edu/rest/fromValues
curl http://cts.fiehnlab.ucdavis.edu/rest/toValues

Expected results:

["ABI Chem","ChEMBL","Chemical Name","InChIKey","MICAD","MLSMR","MMDB","PubChem CID","PubChem SID", ...]
["ABI Chem","ChEMBL","Chemical Name","InChI Code","InChIKey","MICAD","MLSMR","MMDB","PubChem CID","PubChem SID", ...]

Expand formula

Expands a chemical formula.

REST endpoint:

/rest/expandformula/[formula]

Sample call:

curl http://cts.fiehnlab.ucdavis.edu/rest/expandformula/H20

Expected result:

{
    "formula": "H2O",
    "result": "HHO"
}
